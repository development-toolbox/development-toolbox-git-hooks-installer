isort automatically sorts and groups your Python imports for you, keeping your codebase clean and consistent.

üöÄ Quick Start

$ pip install isort
$ isort your_file.py
$ isort . # all files in the project

‚öôÔ∏è GitHub Actions Automation

Automate isort in your CI/CD pipeline! 

# .github/workflows/isort.yml
name: isort check

on: [push, pull_request]

jobs:
  isort:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Set up Python
        uses: actions/setup-python@v4
        with:
          python-version: '3.x'
      - name: Install isort
        run: pip install isort
      - name: Run isort check
        run: isort . --check --diff
  • --check makes isort only report issues, not change files.
  • --diff shows what would change.
Tip: Add isort to your pre-commit config for local automation!
