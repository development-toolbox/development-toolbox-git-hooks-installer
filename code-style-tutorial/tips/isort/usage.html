isort: Usage Patterns & CLI Options

  • Sort a single file:
    $ isort myfile.py
  • Sort all Python files in a directory:
    $ isort .
  • Check for issues without changing files:
    $ isort . --check --diff
  • Use a specific profile (e.g., black):
    $ isort . --profile=black
