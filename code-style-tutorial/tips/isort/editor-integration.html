isort: Editor Integration

  • VS Code: Install the Python extension, then add to settings.json:
    "python.sortImports.args": ["--profile", "black"]
    "editor.codeActionsOnSave": {"source.organizeImports": true}
  • PyCharm: Add isort as an external tool and bind to a shortcut or file save.
  • Other Editors: Most editors support isort via plugins or custom commands.
