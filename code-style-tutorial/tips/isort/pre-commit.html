isort: Pre-commit Hook Example

Add this to your .pre-commit-config.yaml:


-   repo: https://github.com/pycqa/isort
    rev: 5.13.2
    hooks:
      - id: isort
        name: isort (python import sorting)
        entry: isort
        language: python
        types: [python]
        args: [--profile=black]

Then run:

$ pre-commit install
$ pre-commit run --all-files
