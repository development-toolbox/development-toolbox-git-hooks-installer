Python Style: Import Order & Unused Imports

Imports should be grouped: standard library, third-party, then local. Remove unused imports.

‚ùå Bad Example

import os, sys
import requests
from .my_module import foo
import json

def bar():
    print("Hello")
Problem Multiple imports on one line, unused imports, and wrong order.

‚úÖ Good Example

import os
import sys
import json

import requests

from .my_module import foo

def bar():
    print("Hello")
Solution One import per line, grouped and sorted, unused imports removed.

üí° How to Fix with isort

1. Install isort:
$ pip install isort
2. Sort a single file:
$ isort your_file.py
3. Sort all files in your project:
$ isort .
What does isort do?
  • Groups imports into standard library, third-party, and local imports.
  • Sorts each group alphabetically.
  • Ensures one import per line.
  • Removes duplicates and (optionally) unused imports.
