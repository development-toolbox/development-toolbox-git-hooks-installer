Black can be run in your CI pipeline (e.g., GitHub Actions) to enforce formatting.

Add this workflow file to .github/workflows/black.yml:


name: black check

on: [push, pull_request]

jobs:
  black:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Set up Python
        uses: actions/setup-python@v4
        with:
          python-version: '3.x'
      - name: Install black
        run: pip install black
      - name: Run black check
        run: black . --check --diff
