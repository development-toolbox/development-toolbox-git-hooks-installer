Black can be run automatically before every commit using pre-commit.

Add this to your .pre-commit-config.yaml:


-   repo: https://github.com/psf/black
    rev: 24.3.0
    hooks:
      - id: black
        language_version: python3
        args: [--check]
$ pre-commit install
$ pre-commit run --all-files
