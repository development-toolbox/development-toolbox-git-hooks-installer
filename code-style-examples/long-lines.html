Python Style: Long Lines

PEP8 recommends lines be limited to 79 or 88/100 characters. Here’s how to fix long lines:

❌ Bad Example

logger.info("This is a very long log message that goes way beyond the recommended line length and should be split.")

✅ Good Example

logger.info(
    "This is a very long log message that goes way beyond the recommended "
    "line length and should be split."
)

With Function Calls

result = some_function(
    arg1, arg2, arg3,
    arg4, arg5
)
